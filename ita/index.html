Installazione

# Clona il repository
git clone https://github.com/bocaletto-luca/SplitBrowser.git
cd SplitBrowser

# Crea e attiva il virtual environment
python3 -m venv venv
source venv/bin/activate        # macOS / Linux
# .\venv\Scripts\activate.ps1   # Windows PowerShell

# Installa le dipendenze
pip install --upgrade pip
pip install PyQt5 PyQtWebEngine

Uso

# Avvia l'app all'interno del venv
python main.py

Usa il menu o le scorciatoie per gestire schede, split, fullscreen e navigazione.

Scorciatoie da tastiera

  • Ctrl+T: Nuova scheda
  • Ctrl+W: Chiudi scheda
  • F11: Fullscreen intera finestra
  • Shift+F11: Fullscreen riquadro attivo
  • Esc: Esci da fullscreen
  • Ctrl+PgUp / Ctrl+PgDown: Cambia scheda (in fullscreen scheda)
  • Ctrl+Tab / Ctrl+Shift+Tab: Cicla riquadri (in fullscreen riquadro)
