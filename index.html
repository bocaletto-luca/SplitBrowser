SplitBrowser

Dynamic Multi-Pane Web Browser Built with Python & PyQt5

GitHub Repository

Features

  • Multi-tab browsing: open and close tabs easily.
  • Dynamic split layouts: 1, 2, 3 or 4 panes per tab.
  • Equal pane sizing: 50/50, ~33/33/33, 25/25/25/25.
  • Per-pane controls: Back, Forward, Reload, Clear Cache, URL bar.
  • Fullscreen support:
    • Tab fullscreen (F11 / Esc)
    • Pane fullscreen (Shift+F11 / Esc)
  • Keyboard navigation:
    • Ctrl+PgUp / Ctrl+PgDown: switch tabs in tab-fullscreen mode
    • Ctrl+Tab / Ctrl+Shift+Tab: cycle panes in pane-fullscreen mode

Requirements

  • Python 3.7 or newer
  • PyQt5
  • PyQtWebEngine

Installation

# clone the repository
git clone https://github.com/bocaletto-luca/SplitBrowser.git
cd SplitBrowser

# create & activate virtual environment
python3 -m venv venv
source venv/bin/activate        # macOS / Linux
# .\venv\Scripts\activate.ps1   # Windows PowerShell

# install dependencies
pip install --upgrade pip
pip install PyQt5 PyQtWebEngine

Usage

# inside the venv
python main.py

Use the menu or shortcuts to manage tabs, splits, fullscreen, and navigation.

Keyboard Shortcuts

  • Ctrl+T: New Tab
  • Ctrl+W: Close Tab
  • F11: Toggle fullscreen for entire window
  • Shift+F11: Toggle fullscreen for active pane
  • Esc: Exit fullscreen mode
  • Ctrl+PgUp / Ctrl+PgDown: Switch tabs (in fullscreen window)
  • Ctrl+Tab / Ctrl+Shift+Tab: Cycle panes (in fullscreen pane)
